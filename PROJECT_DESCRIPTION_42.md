# 42 專案說明

## Unity 3D 跑酷生存遊戲｜Animal Migration

- 開發玩家控制系統，整合左右移動、跳躍、快速下墜與射擊輸入，並串接動畫狀態切換與死亡流程
- 建立「射擊 → 擊中計分 → 場景持續加速」核心循環，透過動物生成、子彈銷毀與碰撞判定維持遊戲進行
- 實作障礙物與動物隨機生成機制，並依存活時間動態調整場景移動速度與障礙刷新頻率，提升關卡壓力曲線
- 建置遊戲流程管理（開始選單、Game Over、重新開始），完成時間暫停/恢復與重啟後參數重置
- 製作即時 UI 系統，顯示分數與計時器，讓玩家可即時掌握生存表現
- 整合場景與資源，包含地形、植被、特效與音樂素材，並輸出 WebGL 版本供展示與測試
